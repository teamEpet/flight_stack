◆ getEDPReaders()

std::pair<StatefulReader*,StatefulReader*> getEDPReaders ( )

This method returns a pointer to the Endpoint Discovery Protocol Readers (when not in Static mode) SimpleEDP creates two readers, one for Publishers and one for Subscribers, and they are both returned as a std::pair of pointers.

These readers in particular have modified listeners that allow a slave listener to attach its callbach to the original one, allowing for the addition of logging elements.

Returns
std::pair of pointers to the EDP Readers
