Detailed Description

Class HistoryAttributes, to specify the attributes of a WriterHistory or a ReaderHistory.

This class is only intended to be used with the RTPS API. The Publsiher-Subscriber API has other fields to define this values (HistoryQosPolicy and ResourceLimitsQosPolicy).
