◆ defaultOutLocatorList

LocatorList_t defaultOutLocatorList

Default list of Locators used to send messages through.

Used to link with SenderResources in the case and Endpoint is created with NO outLocators. This list contains the default outLocators for the Transports implemented by eProsima.

◆ defaultSendPort

uint32_t defaultSendPort

Default send port that all Endpoints in the RTPSParticipant would use to send messages, default value 10040.

In this release all Endpoints use the same resource (socket) to send messages.

◆ defaultUnicastLocatorList

LocatorList_t defaultUnicastLocatorList

Default list of Unicast Locators to be used for any Endpoint defined inside this RTPSParticipant in the case that it was defined with NO UnicastLocators.

At least ONE locator should be included in this list.
